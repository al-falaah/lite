RTL (Right-to-Left) Text Direction Test

This page tests if Arabic text displays correctly with right-to-left direction.

Test 1: Arabic text WITHOUT dir="auto" (should look wrong)

إِنَّ الَّذِينَ يَتْلُونَ كِتَابَ اللَّهِ وَأَقَامُوا الصَّلَاةَ وَأَنفَقُوا مِمَّا رَزَقْنَاهُمْ سِرًّا وَعَلَانِيَةً يَرْجُونَ تِجَارَةً لَّن تَبُورَ

This should be left-aligned (wrong for Arabic)

Test 2: Arabic text WITH dir="auto" (should be right-aligned)

إِنَّ الَّذِينَ يَتْلُونَ كِتَابَ اللَّهِ وَأَقَامُوا الصَّلَاةَ وَأَنفَقُوا مِمَّا رَزَقْنَاهُمْ سِرًّا وَعَلَانِيَةً يَرْجُونَ تِجَارَةً لَّن تَبُورَ

This should be right-aligned (correct for Arabic)! ✓

Test 3: English text WITH dir="auto" (should be left-aligned)

Indeed, those who recite the Book of Allah, establish the prayer, and spend from what We have provided them—privately and publicly—are hoping for a trade that will never fail.

This English text should be left-aligned (correct) ✓

Test 4: Mixed content (like your blog post)

Allah Almighty says:

إِنَّ الَّذِينَ يَتْلُونَ كِتَابَ اللَّهِ وَأَقَامُوا الصَّلَاةَ وَأَنفَقُوا مِمَّا رَزَقْنَاهُمْ سِرًّا وَعَلَانِيَةً يَرْجُونَ تِجَارَةً لَّن تَبُورَ

"Indeed, those who recite the Book of Allah, establish the prayer, and spend from what We have provided them—privately and publicly—are hoping for a trade that will never fail." [Surah Fāṭir 35:29]

Allah revealed the Qur'an as guidance, light, and mercy for mankind.

What to look for:

  • Test 1 (red border): Arabic should look wrong - starting from left
  • Test 2 (green border): Arabic should be right-aligned and read right-to-left
  • Test 3 (green border): English should be left-aligned and read left-to-right
  • Test 4 (green border): Mixed content with automatic direction switching

If the green-bordered Arabic text is right-aligned, then dir="auto" is working! ✓
